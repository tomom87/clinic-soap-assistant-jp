# 🏥 AI Medical Scribe (医療用AIスクライブ)

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Gemini API](https://img.shields.io/badge/AI-Gemini_2.5_Flash-orange.svg)](https://ai.google.dev/)

**「医師が患者と向き合う時間を最大化する」**

AI Medical Scribeは、診察中の会話音声をリアルタイム（あるいは録音後）に解析し、電子カルテにそのまま貼り付け可能な**SOAP形式**の診療録を自動作成する Chrome 拡張機能です。

---

## 🌟 プロジェクトの目的
日本の医療現場は、過度な事務作業（カルテ入力）によって圧迫されています。
このプロジェクトは、最新のAI技術を活用して「書く」負担を減らし、医師が「診る」ことに集中できる環境をボトムアップで作っていくことを目的としています。

## ✨ 主な機能
- **ワンクリック録音**: ブラウザから直接診察内容を録音。
- **SOAP自動生成**: 日本の医療慣習に即した略語や形式でテキストを構成。
- **Personal Context抽出**: 患者の家族構成や趣味など、信頼関係に欠かせない情報を逃さず記録。
- **APIキー・ローテーション**: 無料枠の制限を回避するため、複数のGemini APIキーを自動切り替え。
- **カスタム・プロンプト**: 診療科や個人のスタイルに合わせてAIの出力をカスタマイズ可能。

## �️ セキュリティとプライバシーについて
本ツールは医療現場での使用を想定しており、プライバシー保護を最優先としています。
- **データ保存**: 音声データや解析結果は、ブラウザを閉じると消去され、サーバー側に蓄積されることはありません。
- **API通信**: ブラウザから直接 Google Gemini API に送信されます。中間サーバーは介在しません。
- **運用の責任**: 利用にあたっては、各医療機関のプライバシーポリシーや患者様の同意に基づき、適切な責任範囲でご利用ください。

## �🛠 セットアップ (開発者向け)
1. 本リポジトリをクローン。
2. Chromeの拡張機能管理画面 (`chrome://extensions`) を開く。
3. 「デベロッパーモード」をONにする。
4. 「パッケージ化されていない拡張機能を読み込む」から、本フォルダを選択。

## 📋 技術スタック
- **Frontend**: HTML, CSS, JavaScript (Vanilla JS)
- **AI Engine**: Google Gemini 2.5 Flash API
- **Audio API**: Web MediaRecorder API
