# 貢献ガイドライン (Contributing to AI Medical Scribe)

本プロジェクトに関心を持っていただきありがとうございます！みなさんの協力が、医療現場の負担軽減につながります。

## 🤝 貢献の方法

### 1. バグ報告・機能提案
- [GitHub Issues](https://github.com/tomom87/clinic-soap-assistant-jp/issues) から報告してください。
- 要望（「この診療科特有のルールを追加したい」など）も大歓迎です。

### 2. プルリクエスト (PR)
1. 本リポジトリを Fork します。
2. 新しい機能や修正のためのブランチを作成します (`git checkout -b feature/amazing-feature`)。
3. 変更をコミットします (`git commit -m 'Add some amazing feature'`)。
4. ブランチを Push します (`git push origin feature/amazing-feature`)。
5. Pull Request を作成してください。

## 💡 重点協力エリア
- **プロンプトの最適化**: 診療科ごとの最適なSOAP抽出プロンプトの作成。
- **UIの改善**: 診察中に邪魔にならない、より直感的なUIへの洗練。
- **セキュリティ・プライバシー**: ローカルでの音声処理やデータの取り扱いに関する知見。

## 💬 コミュニケーション
質問や議論は GitHub Issues または Discussions（有効な場合）で行いましょう。
